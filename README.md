# Linking Naturalistic Emoji Use on Smartphones to Trait and State Affect

This repository contains the analysis code for the paper "Linking Naturalistic Emoji Use on Smartphones to Trait and State Affect".

The project examines how **everyday emoji and emoticon use** is associated with **trait-level affect** (positive and negative affect) and **momentary affective experience** in naturalistic, real-world settings, using smartphone keyboard data.
